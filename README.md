# 💈 Asenso Barber Club

Sitio web desarrollado para una barbería moderna con el objetivo de ofrecer una presencia digital profesional, transmitir confianza y facilitar el acceso a información sobre servicios y estilos de corte.

## 🚀 Demo

https://gonzasenso.github.io/Web-Barberia/

## 🛠 Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript
* Responsive Design
* UI/UX Design

## ✨ Características

* Diseño moderno y atractivo
* Totalmente responsive para dispositivos móviles y escritorio
* Navegación intuitiva
* Sección de servicios
* Galería visual de cortes
* Identidad visual orientada a una barbería profesional
* Optimización para experiencia de usuario

## 📚 Objetivo del Proyecto

Este proyecto fue desarrollado para aplicar conocimientos de desarrollo web frontend, diseño de interfaces y maquetación responsive, simulando un sitio real para un negocio del rubro barbería.
